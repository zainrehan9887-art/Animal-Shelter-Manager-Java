# Animal Shelter Management System (Java)

## 🎯 Overview
A robust Java console application designed to manage animal data using Object-Oriented Programming (OOP). This project was built to practice clean code, encapsulation, and logical data handling.

## 🛠️ Skills Demonstrated
* **OOP Design:** Implementing classes with private attributes and public getters/setters.
* **Logic & Conditionals:** Managing adoption status using boolean flags.
* **Constructors:** Efficiently initializing complex animal objects.
* **Debugging:** Ensuring zero logical errors in data flow.

## 🚀 How to Run
1. Clone the repository.
2. Compile the files: `javac *.java`
3. Run the application: `java Main`
